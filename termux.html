Ikuti langkah-langkah berikut untuk menjalankan script di Termux:

  1. Pastikan Termux sudah terinstall di perangkat Anda.
  2. Buka Termux dan jalankan perintah berikut satu per satu:
# Update dan install dependencies
yes | pkg update && yes | pkg upgrade
pkg install nodejs nano

# Clone script atau buat file manual
git clone https://github.com/user/repo.git
cd repo
nano script.js  # Jika ingin edit script manual

# Install axios
npm install axios

# Jalankan script
node script.js

Pastikan file ua.txt tersedia dalam folder yang sama sebelum menjalankan script.
